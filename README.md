# FB Assignment

A simple front-end assignment demonstrating user signup/login flow with browser localStorage.

## Project Structure

- `index.html` - Landing page showing localStorage examples and sample credentials.
- `Pages/register.html` - Registration page where users can create an account.
- `Pages/login.html` - Login page for authenticating a registered user.
- `Pages/dashboard.html` - Protected dashboard that displays user information after login.
- `Scripts/app.js` - JavaScript logic for signup, login, dashboard data display, and logout.
- `Styles/` - CSS files for styling each page.

## Features

- Create a user account and save details in `localStorage`
- Authenticate users using stored credentials
- Display registered user information on a dashboard
- Protect dashboard access unless the user is logged in
- Logout and clear login state

## Usage

1. Open `index.html` in a browser.
2. Navigate to the registration page and complete the signup form.
3. Login using the registered email and password.
4. View the dashboard and logout when finished.

## Notes

- This app uses browser `localStorage` and is intended for demonstration purposes only.
- Credentials and session state are stored locally and are not secure for production use.
